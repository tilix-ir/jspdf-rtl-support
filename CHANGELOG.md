# Changelog

## 1.1.0
- Underline and strike offsets configurable via constructor (`underlineOffsetRatio`, `strikeOffsetRatio`).
- Underline renders below baseline; strike renders across text body.
- Decoration continuity across segments and spaces.
- Bold, underline, and strike persist across words within tag scope.
- Digit conversion applies only to RTL segments at render time.
- Test render script enhanced for mixed RTL/LTR and styling.
